Phone Division

Mouseenter and Mouseleave Events

Animate and Stop CSS effects on mouseenter and mouseleave events


This is a small paragraph

  1. Mouseenter Event changes opacity from 1 to 0.5, height from 200px to 400px, width from 200px to 400px and fontsize from 12px to 30px
  2. The animation event is for 3 seconds
  3. If mouse leaves the div when mouseover event is runnning, then the animation will stop and starts mouseleave event
  4. If mouse enters the div when mouseleave event is runnning, then the animation will stop and starts mouseenter event

  2. FadeIn 2 secs
